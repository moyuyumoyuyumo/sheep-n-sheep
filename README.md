# 🐑 咩了个咩

> 一个用网页做的三消益智小游戏，玩法借鉴《羊了个羊》。
> 个人学习项目，仅供学习和交流，**非官方、非商用**。

🎮 **在线试玩**：<https://moyuyumoyuyumo.github.io/sheep-n-sheep/> （部署完成后可访问）

---

## 玩法

- 屏幕中间有一堆带图案的方块，叠了很多层（上面的盖住下面的）
- 屏幕下方有 **7 个槽位**
- 点没被压住的方块，方块会飞到下面的槽里
- 槽里**凑齐 3 个一样**的图案就自动消掉
- **槽满 7 个**还没消掉 → 失败
- **把所有方块都消完** → 胜利

## 技术栈

- **HTML5 + CSS3 + 原生 JavaScript** (ES Modules)
- **无构建工具**：浏览器直接打开 `index.html` 即可运行
- **无后端、无账号系统**：玩家存档只存浏览器 `localStorage`
- **托管**：GitHub Pages（免费、永久、HTTPS）

## 本地运行

```bash
# 方式 1：直接打开（最简单）
# 双击 index.html

# 方式 2：用 VS Code 的 Live Server 插件
# 1. VS Code 安装扩展 "Live Server"
# 2. 在 index.html 上右键 → "Open with Live Server"
# 3. 浏览器自动打开，改代码自动刷新
```
